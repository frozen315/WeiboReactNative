一款使用react-native开发的微博客户端 (开发中)

## 开发

### 依赖版本

- Node.js v4.0+
- ReactNative v0.16.0

### 运行

- Clone 项目
- 执行 `npm install`
- 将 `config.example.js` 中的 `app_key` 和 `app_secret` 修改为你自己申请的, 并将文件重命名为 `config.js`
- 在 XCode 中打开 `Yijian.xcodeproj`
- Command + R

### 在设备上运行

1. 执行 `npm run bundle` 生成打包的JS文件
2. Command + R
